MediReturn.co.uk

  • React.js
  • Node.js
  • AWS
  • Google Maps API
  • MongoDB
  • API Integration
  • NodeMailer
  • SMSWorks API

A website to return used crutches to NHS Hospitals, to reduce spending and carbon emissions.
I created this site in React and Node.js, using Google Maps API and other APIs to communicate with various couriers to arrange collection. This site also required learning MongoDB to store user details and consent information, aswell as NodeMailer and SMSWorks APIs for detail authentication. I then contacted key NHS staff, aiming to implement it within NHS University Hospitals Coventry Warwickshire Trust.

Financial Forecasting using Machine Learning

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Scikit-learn
  • Yahoo Finance API
  • Time Series Analysis
  • Statistical Models
  • Machine Learning

For my university dissertation, "Financial Forecasting: Comparing Traditional and Modern Approaches When Forecasting Index Funds", I developed financial forecasting models using statistical and machine learning techniques. I utilised Yahoo Finance to gather historical financial data and applied various models including ARIMA, LSTM, Random Forest, and VARMA. Under the guidance of my supervisor, Dimitar Kazakov, I analysed and compared the performance of these models in predicting stock prices and market trends.
